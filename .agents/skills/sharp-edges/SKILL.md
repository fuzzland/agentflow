# Sharp Edges

Highlight Solidity and EVM edge cases that frequently produce latent vulnerabilities.
Prioritize integer boundary behavior, external call ordering, initialization paths, and upgrade semantics.
Document preconditions that must hold to avoid unsafe execution paths.
