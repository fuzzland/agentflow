# Property Based Testing

Write Solidity invariants and fuzz properties that hold across broad EVM state transitions.
Use Foundry fuzzing and invariant test modes to cover edge inputs, boundary values, and multi-call sequences.
Track failing seeds and reduce to minimal reproductions before proposing fixes.
