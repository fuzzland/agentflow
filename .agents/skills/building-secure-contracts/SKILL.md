# Building Secure Contracts

Constrain guidance to Solidity patterns that reduce EVM risk.
Prioritize checks-effects-interactions, explicit access control, and invariant-driven design for token and vault flows.
Prefer minimal trusted surfaces and explicit failure handling over implicit assumptions.
