# YARA Authoring

Use YARA authoring as compatibility support for mixed artifact investigations.
Scope rule writing to contract audit evidence files, build outputs, and malware-adjacent payloads when relevant.
Do not treat YARA hits as proof without corroborating Solidity or EVM execution evidence.
